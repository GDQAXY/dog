: ["宠物之家"小程序原型设计文档

# 原型设计概览

## 全局导航结构
```
首页（Tab 1）
  ├── 顶部：搜索栏 + 消息中心
  ├── 轮播Banner（5个活动位）
  ├── 8个宫格快捷入口
  ├── 推荐寄养家庭（横向滑动）
  ├── 限时秒杀（商品）
  └── 底部：查看更多

服务（Tab 2）
  ├── 全部服务列表（6大模块）
  └── 最近使用

发布（Tab 3 - 中央按钮）
  ├── 发布寄养需求
  ├── 发布二手商品
  └── 成为养家

消息（Tab 4）
  ├── 系统通知
  ├── 客服消息
  └── 交易消息

我的（Tab 5）
  ├── 用户头像/昵称
  ├── 我的宠物（卡片式）
  ├── 订单中心（各模块订单）
  ├── 我的钱包
  ├── 收藏/足迹
  └── 设置
```

# 一、宠物寄养模块原型

## 1.1 页面清单

| 页面 | 路径 | 说明 |
|------|------|------|
| 寄养首页 | /foster/home | 地图找家+列表 |
| 养家详情 | /foster/host/:id | 养家信息展示 |

| 养家后台 | /foster/host-dashboard | 档期管理+订单 |

# 八、关键交互流程

## 8.1 下单支付流程

```
选择商品/服务
    ↓
填写订单信息（地址、时间、备注）
    ↓
确认订单（费用明细）
    ↓
选择支付方式
    ├── 微信支付
    ├── 支付宝
    └── 余额支付
    ↓
支付成功
    ↓
跳转到订单详情（开始服务流程）
```

## 8.2 消息通知体系

| 场景 | 通知方式 | 内容 |
|------|----------|------|
| 订单状态变更 | 推送+站内 | 您的寄养订单已被接单 |
| 日报更新 | 推送 | 豆豆的今日日报已更新 |
| 聊天消息 | 推送+红点 | 李阿姨发来一条消息 |
| 活动提醒 | 推送 | 您关注的商品正在秒杀 |
| 系统通知 | 站内 | 优惠券即将过期提醒 |

## 8.3 异常状态处理

| 异常 | 处理方式 |
|------|----------|
| 网络断开 | 显示断网提示，自动重试 |
| 支付失败 | 保留订单15分钟，重新支付入口 |
| 定位失败 | 手动输入地址，提示开启定位权限 |
| 上传失败 | 显示重试按钮，保留已选图片 |
| 服务异常 | 友好错误页，联系客服入口 |
