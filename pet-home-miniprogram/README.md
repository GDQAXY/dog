# 宠物之家小程序

根据 HTML 演示文件重新生成的微信小程序项目。

## 项目结构

```
pet-home-miniprogram/
├── app.js                  # 小程序入口逻辑
├── app.json                # 小程序全局配置
├── app.wxss                # 全局样式
├── project.config.json     # 项目配置文件
├── sitemap.json            # 站点地图
└── pages/                  # 页面目录
    ├── index/              # 首页
    │   ├── index.wxml
    │   ├── index.wxss
    │   ├── index.js
    │   └── index.json
    ├── foster/             # 宠物寄养
    ├── medical/            # 宠物医疗
    ├── community/          # 宠物社区
    ├── transport/          # 宠物运输
    ├── shop/               # 宠物商城
    ├── hotel/               # 宠物酒店
    ├── secondhand/         # 二手闲置
    ├── coupon/             # 领券中心
    └── profile/            # 个人中心
```

## 功能模块

### 首页
- 位置选择
- 搜索栏
- 轮播图
- 宫格导航（寄养、医疗、运输、商城、酒店、闲置、领券、社区）
- 我的宠物卡片
- 宠物运输动态
- 信息流（服务、社区帖子、商品）

### 宠物寄养
- 地图模式预览
- 筛选（距离、好评、价格）
- 寄养家庭列表
- 养家详情

### 宠物医疗
- 快捷入口（健康档案、疫苗、驱虫、用药）
- AI智能预诊
- 症状快速选择
- 推荐医生
- 附近医院

### 宠物社区
- 分类标签（推荐、关注、热门等）
- 帖子列表
- 点赞、评论、收藏、分享
- 发布动态

### 宠物运输
- 运输方式筛选（专车、航空、火车、顺风车）
- 路线列表
- 服务保障

### 宠物商城
- 分类筛选（主粮、零食、用品、保健）
- 商品列表
- 购物车

### 宠物酒店
- 酒店列表
- 筛选（距离、好评、价格、豪华）

### 二手闲置
- 分类（用品、食品、求购）
- 闲置列表
- 发布闲置

### 领券中心
- 新人专享券包
- 限时抢券

### 个人中心
- 用户信息
- 我的宠物
- 我的钱包
- 消息中心
- 我的订单
- 我的服务
- 更多功能

## 使用说明

1. 使用微信开发者工具打开项目
2. 在 `project.config.json` 中填入您的小程序 `appid`
3. 编译预览

## 技术栈

- 微信小程序原生开发
- WXML + WXSS + JavaScript
- CSS 变量主题系统

## 注意事项

- 项目中使用的图片资源（icon）需要自行准备，建议使用 emoji 或图标字体代替
- 部分功能为演示效果，实际开发需要接入后端 API
- 地图功能需要申请微信小程序地图权限
