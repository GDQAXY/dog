## 1. 产品概述

### 1.1 产品定位
宠物之家管理后台是为宠物服务小程序提供的一站式运营管理平台，支撑C端用户（宠物主人）与B端服务者（养家、医生、商家）之间的高效撮合，覆盖寄养、医疗、电商、运输等核心业务的运营管理需求。

### 1.2 目标用户
| 角色 | 使用场景 | 核心诉求 |
|------|----------|----------|
| 平台运营人员 | 日常运营、活动配置、内容审核 | 操作便捷、数据可视、流程高效 |
| 财务人员 | 资金结算、对账、退款审核 | 数据准确、流程合规、可追溯 |
| 客服人员 | 处理客诉、订单异常、用户咨询 | 信息完整、响应快速、工具齐全 |
| 管理员 | 权限管理、系统配置、数据监控 | 安全可靠、灵活配置、全局掌控 |

### 2.1 仪表盘

#### 2.1.1 功能描述
提供平台核心数据的实时可视化展示，帮助运营人员快速掌握业务全貌。

#### 2.1.2 页面布局
```
┌─────────────────────────────────────────────────────────────┐
│  今日核心指标                                                │
│  [GMV ¥128,450] [订单数 342] [新用户 89] [活跃用户 1,234]    │
├──────────────────────────────┬──────────────────────────────┤
│  交易趋势（近7天/30天曲线）    │  订单分布（饼图）             │
├──────────────────────────────┼──────────────────────────────┤
│  热门服务TOP10               │  待办事项                     │
│  1. 李阿姨家庭寄养 - 23单    │  ⚠️ 待审核养家 12             │
│  2. 瑞鹏宠物医院 - 18单      │  ⚠️ 待处理退款 5              │
│  ...                         │  ⚠️ 待审核商品 8              │
├──────────────────────────────┴──────────────────────────────┤
│  实时动态                                                   │
│  [14:32] 用户138****1234 下单了「进口猫粮 5kg」¥299        │
```

#### 2.1.3 数据指标定义
| 指标 | 定义 | 更新频率 |
|------|------|----------|
| GMV | 今日成交总额（含待支付） | 实时 |
| 订单数 | 今日下单总量（不含取消） | 实时 |
| 新用户 | 今日注册用户数 | 实时 |
| 活跃用户 | 今日登录用户数 | 实时 |
| 客单价 | GMV / 订单数 | 实时 |
| 转化率 | 下单人数 / 访问人数 | 每小时 |

#### 2.2.2 养家管理（寄养服务者）

**审核流程**
```
提交申请 → 资质初审 → 实地核验 → 培训签约 → 上架服务
    ↑___________________________________________↓
              ←←←← 整改/驳回 ←←←←
```

**列表字段**
| 字段 | 说明 |
|------|------|
| 养家ID | 唯一标识 |
| 姓名/昵称 | 真实姓名+对外昵称 |
| 手机号 | 联系方式 |
| 服务区域 | 省市/商圈 |
| 服务类型 | 狗/猫/异宠 |
| 接单量 | 累计接单数 |
| 评分 | 平均星级 |
| 状态 | 待审核/已通过/已拒绝/已下架 |

**详情页字段**
- 个人信息：身份证正反面、手持身份证照、居住证明
- 居住环境：房屋面积、是否有院子、是否有原住民宠物
- 服务配置：可接待宠物类型、最大同时接待数、价格设置
- 相册：环境照片（至少5张）
- 审核记录：历史审核意见

#### 2.2.3 医生管理（医疗服务者）

**审核流程**
```
提交申请 → 资质审核 → 视频面试 → 签约入驻 → 开通服务
    ↑______________________________↓
           ←←←← 整改/驳回 ←←←←
```

**列表字段**
| 字段 | 说明 |
|------|------|
| 医生ID | 唯一标识 |
| 姓名 | 真实姓名 |
| 执业医院 | 所属宠物医院 |
| 职称 | 兽医师/高级兽医师/专家 |
| 擅长领域 | 内科/外科/皮肤科/骨科等 |
| 从业年限 | 工作年限 |
| 资质状态 | 已上传/审核中/已通过/已驳回 |
| 服务状态 | 在线/离线/休息中 |
| 接诊数 | 累计接诊次数 |
| 评分 | 用户评分 |
| 操作 | 查看详情/资质审核/状态管理 |

**资质审核要点**
- 执业兽医资格证
- 学历证书
- 工作证明
- 所属医院营业执照（如适用）
- 专业培训证书（如有）

**服务配置**
- 在线咨询价格/时长
- 图文咨询价格
- 视频咨询价格
- 可预约时间段
- 擅长病种标签
